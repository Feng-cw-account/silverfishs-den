# 100 Zettelkasten 工作流 (CODE)

这一套流程旨在将外部信息转化为你自己的知识。

## 1. 捕获 (Capture) - 收集种子
*   **场景**：阅读、看视频、闪念。
*   **动作**：在 `00_Inbox` 新建笔记。
*   **标签**：`#type/fleeting` 或 `#type/lit`，加上 `#status/seed`。
*   **内容**：快速记录，不求完美。

## 2. 提炼 (Distill) - 原子化转译
*   **场景**：每日或每周复盘。
*   **核心动作**：
    1.  回顾 Inbox 里的种子。
    2.  **用自己的话**重写核心观点。
    3.  如果观点包含多个主题，**拆分**成多条笔记。
    4.  **打标**：修改为 `#type/permanent`，属性改为 `#attr/concept` 或 `#attr/principle`。
    5.  **移动**：拖入 `10_Zettelkasten` 文件夹。

## 3. 连接 (Connect) - 织网
*   **场景**：在写永久笔记时。
*   **思考**：
    - "这个观点和库里已有的什么观点相似？" -> 建立类比链接
    - "这个观点支持/反驳了什么？" -> 建立辩证链接
*   **动作**：使用 `[[笔记名]]` 建立双向链接。

## 4. 表达 (Express) - 产出
*   **场景**：需要写作、做项目。
*   **动作**：创建 `#type/moc`，将相关原子笔记串联起来，形成文章骨架。
